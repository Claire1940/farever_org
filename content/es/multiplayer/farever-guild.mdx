<Section icon={ShoppingBag} title="El Mercader del Gremio Farever y el Inventario">

Aunque un sistema completo de **gremio Farever** dirigido por jugadores está en desarrollo, el Mercader del Gremio actualmente sirve como el NPC más importante para el mantenimiento del personaje. Este mercader es tu fuente principal para mejoras de mochila, las cuales son esenciales dada la gran cantidad de botín que sueltan los jefes.

<Steps>
  <Step num="1" title="Recolectar Oro">
    Completa objetivos del mapa y mazmorras para acumular oro. También puedes hacer clic derecho en "Paquetes Perdidos" en tu inventario para convertirlos en "Paquetes Completados" por 100 de oro cada uno en el mercader.
  </Step>
  <Step num="2" title="Comprar Bolsas">
    Visita al Mercader del Gremio y selecciona "Comerciar". Compra las Bolsas Pequeñas (6 ranuras cada una) para llenar tus ranuras de mochila vacías.
  </Step>
  <Step num="3" title="Comprar Herramientas de Utilidad">
    El mercader también vende Viales para Alquimia y Redes para atrapar mascotas. Siempre ten una pila de Viales para crear pociones curativas sobre la marcha.
  </Step>
</Steps>

<Section icon={Swords} title="Sistemas de Combate: Armas, Arsenal y Runas">

El combate en Farever se divide entre las habilidades de tu clase y las habilidades de tu arma. Cada clase puede equipar múltiples tipos de armas, cada una otorgando un conjunto único de habilidades activas y pasivas. Actualmente, hay más de 25 armas disponibles, lo que permite una diversidad significativa de builds incluso dentro de una misma clase.

| Sistema | Función | Cómo Optimizar |
| :--- | :--- | :--- |
| **Habilidades de Armas** | Otorga 2-4 habilidades únicas | Sube de nivel las habilidades con signos "+" verdes |
| **Ranura de Arsenal** | Habilidad pasiva de una segunda arma | Equipa un bastón de curación para mantenerte |
| **Runas** | Modifica el comportamiento de las habilidades de clase | Haz clic derecho en las runas para desbloquear nuevos rasgos |
| **Mejora** | Pequeñas mejoras de estadísticas (+1 Fe, etc.) | Guarda materiales para equipo de alto nivel (Nivel 20) |

La ranura de **Arma de Arsenal** es una mecánica única que te permite "tomar prestada" una habilidad de un arma que no estás sosteniendo activamente. Por ejemplo, un Pícaro podría equipar un cetro curativo en la ranura de Arsenal para obtener un escudo periódico o regeneración de salud.

<Callout type="success" title="Sinergia de Habilidades">
Al subir de nivel las habilidades de armas, busca sinergias. Algunas habilidades infligen un 50% más de daño a enemigos "ralentizados". Combinar esto con un arma que aplica una desventaja de movimiento es clave para un alto DPS.
</Callout>

</Section>

<Section icon={Sparkles} title="Crafteo y Profesiones">

Las profesiones, o "Trabajos", son esenciales para la viabilidad en el endgame. Si bien puedes aprenderlas todas, el costo de desbloquear cada profesión subsiguiente aumenta. Para los principiantes, la **Alquimia** es el punto de partida recomendado debido a la necesidad de autocuración.

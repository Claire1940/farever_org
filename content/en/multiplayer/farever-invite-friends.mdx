<Section icon={Settings} title="Troubleshooting Multiplayer Issues">

Even in 2026, network issues can occasionally prevent you from being able to **farever invite friends**. Most problems in Farever's Early Access phase stem from mismatched settings or minor server desyncs.

| Problem | Potential Cause | Solution |
| :--- | :--- | :--- |
| **Invite not received** | Private Steam Profile | Set profile to 'Friends Only' or 'Public' |
| **Join Failed error** | Version mismatch | Restart Steam to force an update check |
| **High Latency (Lag)** | Wrong server region | Check 'Settings' > 'Network' for region |
| **Cannot see friend** | Different world instance | Use the 'Teleport to Leader' function |

<Steps>
  <Step num="1" title="Check the Official Server Status">
    Visit the Shiro Games Twitter (X) or the official Discord to see if there is scheduled maintenance or a known server outage.
  </Step>
  <Step num="2" title="Verify Game Files">
    If the game crashes upon joining a friend, right-click Farever in Steam > Properties > Installed Files > **Verify integrity of game files**.
  </Step>
  <Step num="3" title="Sync System Clocks">
    Ensure your Windows/PC clock is set to 'Automatic'. Mismatched timestamps can sometimes interfere with Steam's authentication tokens.
  </Step>
</Steps>

<Callout type="warning" title="Anti-Virus/Firewall">
Ensure that Farever.exe is added to your firewall's whitelist. Some aggressive security software may block the peer-to-peer packets required for the Steam invitation system to handshake correctly.
</Callout>

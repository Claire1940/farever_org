最初に出くわす攻撃は**Tidal Slash**です。この攻撃は4.5秒の詠唱時間があり、これがダメージを与える主要なウィンドウとなります。ボスがこの技をチャージしている間は、ハイライトされた攻撃範囲外にいれば比較的安全です。

| 攻撃名 | 詠唱時間 | 危険度 | 説明 |
| :--- | :--- | :--- | :--- |
| **Tidal Slash** | 4.5秒 | 中 | 広範囲の弧状スラッシュ。プレイヤーがDPSを出すかリソースを回復するのに最適な時間。 |
| **Devour** | 10秒 | **致命的** | 即死ダッシュ。ボスが「空腹」になりプレイヤーを追いかける。 |
| **Nia's Fury** | 10秒 | **致死的** | アリーナ全体を覆う洪水と波。生き残るには垂直方向の位置取りが必要。 |
| **Minion Summon** | 即時 | 低/高 | 画面を埋め尽くし、移動を妨げる小さなカニを召喚する。 |
| **Dash Strike** | 2秒 | 中 | ローテーションをリセットするためにアリーナの中央に素早く移動する。 |

<Callout type="warning" title="危険: Devourメカニズム">
ボスが**Devour**を準備したら、すぐにすべての攻撃的な行動を停止してください。この攻撃は10秒間続き、ボスに捕まると即死します。ダッシュ、回避、移動アビリティを使用して最大距離を維持してください。
</Callout>

<Section icon={Target} title="Nia's Furyと洪水を生き残る">

**Farever Crabgantua**戦で最も複雑な部分は、**Nia's Fury**への移行です。ボスがTidal SlashとDevourのサイクルを完了した後、アリーナの中央にダッシュします。これは10秒間のチャージ時間の始まりを示し、その間、床全体が死の罠となります。

アリーナは水で満たされ、生き残る唯一の方法は環境を有利に利用することです。地面に青い円が現れ、キノコの足場が落ちてくる場所を示します。同時に、水しぶきが噴出し、一時的なエレベーターとして機能します。

| 生存要素 | 機能 | タイミングの注意点 |
| :--- | :--- | :--- |
| **水しぶき** | プレイヤーを空中に打ち上げる | 早すぎるとジャンプしないでください。水しぶきは1秒で消えます。 |
| **キノコの足場** | 安全な立ち場を提供する | 地面の青い円を見て、その位置を予測してください。 |
| **メイジのBlink** | 波をテレポートで通過する | 水しぶきのタイミングを逃した場合の最終手段として使用できます。 |

<Steps>
  <Step num="1" title="ダッシュを特定する">
    クラブガンチュアがアリーナの厳密な中央に移動するのを観察してください。この特定の動きをするのはこの時だけです。
  </Step>
  <Step num="2" title="水しぶきを見つける">
    最寄りの水しぶきを見つけ、その近くに位置しますが、まだその上に直接立たないでください。
  </Step>
  <Step num="3" title="2秒ルール">
    Nia's Furyの詠唱バーが2秒から1.5秒の間になるまで待ちます。この時点で水しぶきにジャンプして最大高度に到達します。
  </Step>
  <Step num="4" title="キノコに着地する">
    落下するキノコの足場に向かってキャラクターを誘導します。波が通過する前に地面に落ちると死亡します。
  </Step>
</Steps>

<Callout type="tip" title="プロのタイミング">
水しぶきに早すぎるとジャンプすると、致命的な波が放たれるのと同時に地面に落下してしまいます。タイマーがゼロになったときに、ジャンプの最高点にいるか、キノコの上に安全に着地していることを目指しましょう。
</Callout>

</Section>

**Farever Crabgantua**へのアプローチ方法は、選択したクラスに大きく依存します。近接キャラクターはTidal Slash中に位置取りに細心の注意を払う必要がありますが、遠隔キャラクター（特にメイジ）は安全性とユーティリティの点で明確な利点があります。

  <Card title="メイジの利点">

    - **レンジ**: Devourフェーズ中に「ポットショット」を放つことができます。
    - **シールド**: スパーク回復を使用してシールドをアクティブに保ちます。
  </Card>
  <Card title="近接戦術">

  </Card>
  <Card title="群衆制御">

    - **燃焼効果**: カイトしながらボスに継続ダメージを与えます。
  </Card>

戦闘中、クラブガンチュアはプレイヤーを圧倒する可能性のあるミニオンを召喚します。これらの小さなカニは個々には大きなダメージを与えませんが、あなたを隅に閉じ込め、ボスの即死攻撃の簡単な標的にする可能性があります。

| 戦略の種類 | 推奨スキル | 有効性 |
| :--- | :--- | :--- |
| **直接ダメージ** | ファイアボール / パワークラズム | 高 |
| **群衆制御** | 火炎放射器 / AoE | 必須 |
| **リソース獲得** | スパーク / 基本攻撃 | 常に |
| **機動力** | ブリンク / ダッシュ | 致命的 |

<Callout type="success" title="メイジ戦略">
メイジは**スパーク**の回復を高く保つことに集中すべきです。**火炎放射器**を使ってミニオンの群れを溶かし、**ファイアボール**でボスに一貫したダメージを与えます。波の間に位置を失った場合は、**ブリンク**のタイミングを合わせて水を通り抜けましょう。

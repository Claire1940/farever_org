<Section icon={Map} title="はじめに：無料マウントと探索">

Fareverの世界に入ってすべてのプレイヤーが最初に行うべき行動は、移動手段の確保です。高レベルのクエストや課金通貨の背後にマウントを隠している従来のMMOとは異なり、Fareverはスムーズな探索を保証するためにスターターセットを無料で提供しています。

**動画のハイライト:**
- ショップで無料スターターパックを受け取る方法。
- コレクション（Y）メニューからマウントを装備する方法。
- グライディング（滑空）とライディング（騎乗）の切り替えについて。
- 騎乗用のカスタムホットキーの設定。

マウントを手に入れたら、主な焦点はマップの完了に移ります。世界は「未探索領域（グレーエリア）」と「クエスチョンマーク」で満たされた地域に分かれています。これらの目標をクリアすることが、XPを獲得する最速の方法です。

| 目的の種類 | 報酬の種類 | 優先度 |
| :--- | :--- | :--- |
| **オベリスクの起動** | ファストトラベル＆マップ解放 | 最優先 |

| **コーデックス狩り** | ステータス上昇＆ボーナスXP | 中 |
| **ダンジョン入場** | ルーン＆レア装備 | 高 |

<Callout type="tip" title="ファストトラベルのヒント">

<Section icon={ShoppingBag} title="Farever ギルド商人とインベントリ">

プレイヤー運営の本格的な**Farever ギルド**システムは現在開発中ですが、ギルド商人は現在、キャラクターのメンテナンスにおいて最も重要なNPCとして機能しています。この商人は、ボスからドロップする膨大な量の戦利品を考慮すると不可欠な、バッグのアップグレードの主な入手先です。

<Steps>
  <Step num="1" title="ゴールドを集める">
    マップの目標やダンジョンを完了してゴールドを蓄積します。インベントリにある「Lost Packages（紛失した荷物）」を右クリックして「Completed Packages（完了した荷物）」に変換し、商人で1つ100ゴールドで売ることもできます。
  </Step>
  <Step num="2" title="ポーチを購入する">
    ギルド商人を訪ね、「Trade（取引）」を選択します。Small Pouches（各6スロット）を購入して、空いているバッグスロットを埋めましょう。
  </Step>
  <Step num="3" title="ユーティリティツールを買う">
    商人は錬金術用の「Vials（瓶）」や、ペットを捕まえるための「Nets（網）」も販売しています。外出先で回復ポーションを製作できるよう、常にVialsのスタックを持ち歩きましょう。
  </Step>
</Steps>

<CardGrid cols={2}>
  <Card title="インベントリ管理">

    - **銀行へのアクセス**: 訓練マニュアルやレア素材を保管。
    - **パッケージ取引**: ジャンクアイテムを100gの報酬に変換。

  <Card title="ユーティリティの購入">

  </Card>
</CardGrid>

</Callout>

<Section icon={Swords} title="戦闘システム：武器、アーセナル、ルーン">

Fareverの戦闘は、クラススキルと武器スキルの2つに分かれています。すべてのクラスが複数の武器タイプを装備でき、それぞれが独自のセットのアクティブおよびパッシブアビリティを付与します。現在、25種類以上の武器が利用可能で、単一のクラス内でも多様なビルドが可能です。

| システム | 機能 | 最適化の方法 |
| :--- | :--- | :--- |
| **武器スキル** | 2～4個の固有アビリティを付与 | 緑色の「+」サインでスキルをレベルアップ |
| **アーセナルスロット** | 2つ目の武器からのパッシブスキル | 維持力を高めるために回復の杖を装備 |
| **ルーン** | クラススキルの挙動を変更 | ルーンを右クリックして新しい特性を解放 |
| **アップグレード** | わずかなステータス上昇（信仰+1など） | 高レベル（Lv.20）装備のために素材を温存 |

**アーセナル武器**スロットは、現在アクティブに持っていない武器から1つのスキルを「借りる」ことができるユニークなメカニクスです。例えば、ローグがアーセナルスロットに回復のセプターを装備して、定期的なシールドや体力再生を得ることができます。

<Callout type="success" title="スキルの相乗効果">
武器スキルを上げる際は、相乗効果（シナジー）を探しましょう。一部のスキルは「スロウ（鈍足）」状態の敵に対してダメージが50%増加します。これを移動デバフを与える武器と組み合わせることが、高いDPSを出す鍵となります。
</Callout>

</Section>

専門職、いわゆる「ジョブ」は、エンドゲームで活躍するために不可欠です。すべてのジョブを学ぶことができますが、新しいジョブを解放するたびにコストが増加します。初心者には、自己回復の必要性から**錬金術**を最初に始めることをお勧めします。

料理も同様に重要ですが、スカンクシチューのような価値の高い食品を作るには「製作ポイント」が必要です。これらのポイントは時間とともに回復するため、何を製作するか慎重に選ぶ必要があります。

| 専門職 | 主な成果物 | 材料の入手先 |
| :--- | :--- | :--- |
| **錬金術** | 体力＆マナポーション | ハーブ ＆ Vials |
| **料理** | ステータスバフ料理 | モンスターの肉 ＆ ソース |
| **鍛冶** | 防具 ＆ 武器のアップグレード | 分解されたスパーク ＆ 鉱石 |
| **狩猟** | 皮革 ＆ 特殊素材 | 動物型モンスター |

<Callout type="info" title="水中メカニクス">
Fareverには泳ぐアニメーションがなく、海底を歩きます。2段ジャンプを使って移動し、青い呼吸バーに注意してください。回復ポーションを使うことで、長い水中セクションを「力技」で突破することも可能です。

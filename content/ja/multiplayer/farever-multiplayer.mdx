## 戦闘クラスと武器ベースのスキルシステム

Fareverでは、プレイスタイルは選択したクラスと装備する武器の組み合わせによって決まります。最初は4つの基本アーキタイプから選びますが、**Fareverのマルチプレイヤー**における戦闘の真の深みは、世界中に存在する100種類以上のユニークな武器にあります。

### 基本クラスアーキタイプ
1.  **ウォリアー：** 前衛のタンク兼ブロウラー。高いヘルス、ガード、物理ダメージに特化。
2.  **クレリック：** 多彩なサポートクラス。セプターを使った伝統的なヒーラーや、トーテムを使った「シャーマン」スタイルのダメージディーラーとしても構築可能。
3.  **ローグ：** 高機動DPS。「ジッピー」な移動スキルとシャドウフィールドを使い、敵を翻弄する。
4.  **メイジ：** 遠距離属性ダメージ。ブリンクによる位置取りや強力な範囲攻撃（AoE）魔法を活用する。

> **💡 ヒント：** すべての武器には固有のコンボフィニッシャーがあります。基本攻撃の最中にパーフェクトガードを決め、そのままコンボを完遂することで絶大なボーナスダメージを与えることが可能です。

### アーセナル（Arsenal）システム
最も革新的な機能の一つが「アーセナル」です。レベル7になるとサブ武器を装備できるようになります。これは単なるステータスアップではなく、その武器から特定のスキル（アクティブまたはパッシブ）を引き出し、メインのスキルバーに追加できるというものです。例えば、両手斧を使うウォリアーがアーセナルに盾を装備すれば、「シールドバッシュ」によるスタンが使用可能になり、グループプレイでの有用性が大幅に向上します。

## オープンワールドの探索：ダンジョンとエリート

Fareverの世界には戦闘の機会が溢れています。クエストハブを巡るだけの一般的なMMOとは異なり、Fareverは探索に重きを置いた「テーマパーク」のような作りです。プレイヤーを導くようなメインストーリークエストは存在しません。マップの隅々まで探索することで、自らコンテンツを見つけ出すのです。

### ワールドボスと「シャイニー」
各地域には「エリート」と呼ばれるワールドボスが複数存在します。丘の上にいる巨大なスライム「Blob Dylan」や、宝箱を守る高レベルのエレメンタルなどがその例です。**Fareverのマルチプレイヤー**において、これらはパブリックイベントです。誰かがボスと戦っているのを見かけたら、迷わず参加しましょう。ルートは個別かつ共有されるため、見知らぬ人を助けることにデメリットはありません。また、貴重なクラフト素材や装備をドロップする、少し倒しやすいエリートモンスター「シャイニー」や「スパークリー」も存在します。

### インスタンスダンジョン
主要な各地域には、最低1つは専用のインスタンスダンジョンが用意されています。これらは高ティアの「青」レアリティ装備の主な入手源です。
*   **レイアウト：** ダンジョンごとに独自のトラバーサル（移動）パズルや敵の配置が設定されています。
*   **ボスギミック：** ボスには複数のフェーズがあり、ボスの位置をコーナーに誘導して炎を管理するなど、特定の攻略メカニズムが求められます。
*   **リプレイ性：** ダンジョンは繰り返し挑戦可能です。ビルドを極めたいプレイヤーにとっての主要なアクティビティとなります。

| アクティビティの種類 | 報酬の可能性 | マルチプレイヤースタイル |
| :--- | :--- | :--- |
| **ワールドボス** | 高（装備＆素材） | オープンワールド、共有参加型 |
| **インスタンスダンジョン** | 非常に高（レア武器） | グループベース、プライベートインスタンス |

| **タイムトライアル** | 中（コスメティック＆装備） | 個人のスキル挑戦型 |

## トラバーサルと移動メカニクス

Fareverの移動は高速かつ流れるようで、世界の立体的な地形を駆け回るのが楽しくなるように設計されています。プレイヤーは**Fareverのマルチプレイヤー**の旅の開始直後から、いくつかの移動ツールを利用可能です。

*   **グライダー：** 高所からいつでも滑空可能。重要な点として、グライダーはスタミナを消費しないため、長距離移動に適しています。
*   **グラップルフック：** 世界中に点在するグラップルポイントを利用して、隙間を飛び越えたり、勢いをつけて「ヨーヨー」のように移動したりできます。
*   **ランチパッド：** 空中に打ち上げられ、グライダーと組み合わせることで隠されたエリアに到達できます。
*   **クライミングパズル：** 螺旋塔のようなエリアでは、チェックポイントやミニボスが存在する、多段階のクライミングチャレンジを攻略する必要があります。

## 成長システム：レベル、ジョブ、ルーン

Fareverの成長要素は多層的です。キャラクターのレベルだけでなく、使用する武器ごとに習熟度を上げていく必要があります。

1.  **キャラクターレベル：** 基本クラスのパッシブスキルやアクティブスキルの解放。
2.  **武器レベル：** 特定の武器を使用するとスキルポイントを獲得。これを使用して、その武器固有のアビリティを強化できます（例：パーフェクトガード後のシールドバッシュダメージを100%増加させるなど）。
3.  **ジョブ（職業）：** 鍛冶などの役割を選択可能。武器を3つのティアにアップグレードでき、ステータス強化や強力な最終ティアのパッシブ効果が解放されます。
4.  **ルーンシステム：** クラススキルをさらにカスタマイズ可能。例えば、「チャージ」スキルに「不安定のルーン」を装着することで、チャージでヒットした敵のダメージを減少させるといった効果を付与できます。

| 成長レイヤー | 主な恩恵 | 進行方法 |
| :--- | :--- | :--- |
| **キャラクターXP** | クラススキル＆基本ステータス | クエスト、戦闘、探索 |
| **武器習熟度** | 武器スキルの強化 | 特定の武器でのダメージ蓄積 |
| **鍛冶** | ステータスアップ＆パッシブ | 資源収集とクラフトステーションでの作業 |
| **ルーンソケット** | スキルの特化 | ダンジョンやワールド宝箱からの入手 |

## FAQ

**Q：Fareverは画面上に何千人ものプレイヤーがいる伝統的なMMOですか？**

A：Fareverは「MMOライト」と呼ぶのが最適です。多くのプレイヤーが見える共有世界ではありますが、シャーディングを使用してプレイヤー密度を調整しており、混雑しすぎず冒険感を感じられるレベルに保たれています。

**Q：Fareverのマルチプレイヤーはソロで遊べますか？**

A：はい。ワールドボスや移動パズルを含むほとんどのコンテンツを一人でクリア可能です。ただし、世界には他のプレイヤーが存在しており、ダンジョンに関しては最高効率のルートを入手するためにグループを組むことが推奨されます。

**Q：クラスの新しいスキルはどうやってアンロックしますか？**

A：スキルは2つの方法でアンロックされます。キャラクターレベルを上げてクラス固有のアビリティを獲得する方法と、武器レベルを上げてその武器に紐付いたスキルを解放する方法です。

**Q：グライダーにスタミナは必要ですか？**

A：いいえ、Fareverのグライダーはスタミナバーを消費しません。高度が許す限りどこまでも滑空でき、これが本作の自由な探索体験の重要な要素となっています。

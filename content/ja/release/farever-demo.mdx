<Section icon={Shield} title="戦闘メカニクスとアーセナルシステム">

Fareverの戦闘は、単なるボタン連打以上のものを必要とするダイナミックなものです。**farever demo**版をプレイしている場合でも、フル早期アクセスビルドをプレイしている場合でも、「パーフェクトブロック」と「スタミナ管理」のリズムをマスターすることが、ボス戦を生き抜く鍵となります。

| メカニクス | 説明 | プレイヤーのメリット |
| :--- | :--- | :--- |
| **パーフェクトブロック** | 攻撃が当たる瞬間に正確にブロックする。 | スタミナを消費せずにダメージを無効化する。 |
| **アーセナルスロット** | サブ武器のスキルを装備する。 | 斧を使いながら弓のスキルを使用できる。 |
| **スタミナ** | 回避やブロックで消費されるリソース。 | これを管理することで、気絶や無防備状態を防ぐ。 |
| **スキルチェイン** | クラスアビリティを武器の動きと連続して使用する。 | 敵の「スタッガー（怯み）」中にDPSを最大化する。 |

<Callout type="warning" title="スタミナの警告">
ボス戦中にスタミナバーを完全に使い果たさないでください。スタミナがゼロの状態で捕まると、ボスのアルティメット攻撃を回避できず、通常は即座にノックアウトされます。
</Callout>

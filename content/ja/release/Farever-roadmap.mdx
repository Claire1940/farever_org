<Section icon={Zap} title="PvPと競技プレイ">

競争好きなプレイヤーには多くの楽しみが待っています。**Fareverロードマップ**では、純粋なPvE体験からバランスの取れたPvP環境への移行が明示的に詳述されています。これには、PvEバランスを崩さない専用の装備が含まれます。

**計画されているPvP機能:**
- **決闘**: オープンワールドのハブでプレイヤーに練習を挑む。
- **PvP地域**: 常時戦闘が可能なハイリスク・ハイリターンのゾーン。
- **インスタンス化された戦場**: 目標ベースの10対10または20対20のアリーナ。
- **専用のバランス調整**: 公平なプレイを保証するために、PvP用の個別のスキル係数。

| PvPモード | 報酬の種類 | プレイヤー数 |
| :--- | :--- | :--- |
| **決闘** | 自慢の権利 | 1対1 |
| **戦場** | PvP通貨/装備 | 10対10以上 |
| **オープンワールドゾーン** | 希少なクラフト材料 | FFA / グループ |
| **ランクアリーナ** | シーズンタイトル | 2対2 / 3対3 |

<Callout type="info" title="PvPバランス調整">
開発者は、PvP装備には他のプレイヤーとの戦闘でのみ発動する特定のステータスがあり、PvEダンジョンでの「パワーインフレ」を防ぐと述べています。
</Callout>

</Section>

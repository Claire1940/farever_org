## クラスシステムと初期の戦闘メカニクス

Fareverはリアルタイムのアクション戦闘システムを採用しています。つまり、タブターゲット方式ではなく、手動で攻撃を狙い、敵のダメージを避けるために回避のタイミングを計る必要があります。現在、ゲームには4つの基本アーキタイプがありますが、「アーセナル（Arsenal）」システムにより、装備する武器に基づいてプレイスタイルを大幅に変化させることができます。

| クラス | 主な役割 | 主要メカニクス |
| :--- | :--- | :--- |
| **ウォリアー** | タンク / 近接DPS | レイジ/バーサクゲージを使用して、強力な打撃やシールドバッシュを強化する。 |
| **メイジ** | 遠距離DPS | 属性魔法の範囲攻撃（AoE）とマナ管理に焦点を当てる。 |

戦闘は流動的で、「ガード（Block）」と「回避（Dodge）」のメカニクスをマスターしたプレイヤーが報われる仕組みです。ガードですべてのダメージを軽減できるわけではありませんが、エリートモンスターやワールドボスとの戦いでは不可欠です。

## 武器スキルシステム（アーセナル）

**farever beta access**のプレイテストで明らかになった最も興味深い機能の一つが、「アーセナル（Arsenal）」システムです。『Guild Wars 2』のように、使用できるスキルは現在装備している武器によって決まります。例えば、剣と盾を使うウォリアーは防御的な能力を持ちますが、双斧に切り替えるとホットバーが完全に変化し、高速な範囲攻撃（AoE）に特化します。

これらのスキルを使用すると、個別にランクが上がります。キャラクターのメインレベルが一定期間変わらなくても、「プラクティスストライク」や「シールドバッシュ」のレベルを上げることで、ダメージを増やしたりクールタイムを短縮したりできます。これにより、特定の武器タイプに特化できる「水平プログレッション（横の成長）」の要素が加わっています。

## 専門職と採集

Fareverは戦闘だけではありません。高レベルの進行に欠かせない堅牢な専門職（プロフェッション）システムを備えています。現在のビルドでは、プレイヤーは主要な町のトレーナーを訪ねて「ジョブ（職業）」を学ぶことができます。しかし、一度に集中できる主要な専門職は通常1つだけという制限があり、プレイヤー間の取引やギルドの協力を促しています。

### 2026年に利用可能な専門職

| 専門職 | 主な機能 | 主要材料 |
| :--- | :--- | :--- |
| **鍛冶** | 重装鎧や近接武器を製作。 | 銅鉱石、鉄、石炭。 |
| **錬金術** | ポーション、エリクサー、戦闘バフを作成。 | 薬草、モンスターの精髄。 |
| **裁縫** | 軽装・中装鎧の仕立て。 | 動物の毛皮、シルク、糸。 |
| **宝石細工** | 指輪、ネックレス、ステータス強化の宝石を製作。 | 貴石、銀。 |
| **料理** | 長期的なステータスボーナスを与える料理を作る。 | 肉、野菜、スパイス。 |

採集はアクティブなプロセスです。銅や鉄を採掘するには、つるはしを装備し、世界中に点在するノードを見つける必要があります。これらのノードからは原石のほか、さまざまな二次製作レシピで使用される「ひび割れた石（Cracked Rock）」が手に入ることがあります。

> **💡 ヒント：** 旅の早い段階で2番目の町をチェックしましょう。そこの製作トレーナーは、専門職としてのキャリアをスタートさせるために必要な道具と初期クエストを提供してくれます。

## 探索とワールドイベント

Fareverの世界はゾーンに分かれており、プレイヤーは特定のテレポート地点や展望ポイントを訪れることでこれらを「アンロック」できます。ゾーンがアンロックされると、マップUIにさまざまなクエスト、重要地点、そして「コーデックス（Codex）」の項目が表示されます。

コーデックスは、**farever beta access**を持つプレイヤーにとって不可欠なツールです。倒したすべてのモブを追跡し、潜在的なドロップアイテムや製作材料の収穫量を表示します。特定の地域のコーデックスを完了すると、経験値やユニークな装備が報酬として与えられることがよくあります。

### エリートキャンプとダンジョン
フィールドの至る所で「エリートキャンプ」に遭遇します。これらは通常のモブよりも大幅にタフな「がっしりした」NPCによって示されます。これらのキャンプを掃討すると、青ティア（レア）装備が入った宝箱を獲得できることが多いです。

より大きな挑戦を求めるプレイヤーのために、インスタンスダンジョンも用意されています：
*   **レベル5ダンジョン：** グループメカニクスの基本を学ぶ入門用。モブのヘイトを効果的に管理するために、最低2人のプレイヤーが必要です。
*   **レベル7ダンジョン：** 4人のフルパーティ推奨。ボスのギミックとスキルの連携に焦点を当てています。

## FAQ

**Q: Fareverのベータアクセスを入手するにはどうすればよいですか？**

**Q: Fareverはタブターゲット方式ですか、それともアクション戦闘ですか？**

**Q: Fareverでクラスを変更することはできますか？**

**Q: ベータ版にマイクロトランザクション（課金）はありますか？**

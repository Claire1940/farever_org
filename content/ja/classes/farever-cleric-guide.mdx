ハイリスクなアクションRPGでサポート役を選ぶには、生存能力とユーティリティのバランスが重要です。この**Farever クレリックガイド**は、見習い信徒から神聖なる介入の達人へと成長するための手引書です。後方に控える従来のMMOのヒーラーとは異なり、この**Farever クレリックガイド**では、回復量と同じくらいポジショニングとタイミングが重要な、本作独自の「ソウルライク」な戦闘に焦点を当てています。Fareverのクレリックは単に傷を癒やすだけでなく、物理的な当たり判定やアクティブな軽減手段を駆使して、聖なるダメージを与えながら仲間を守るために戦いの最前線に立ちます。

## クレリックの基本戦闘メカニクス

クレリックとして成功するには、まずFareverの基本的な戦闘システムをマスターする必要があります。このゲームでは、従来のマナプールの代わりにスタミナベースの防御システムを採用しています。クレリックにとって、クールダウンとブロックゲージの管理が生存を分ける鍵となります。

### 防御アクション
すべてのクレリックは、基本的なブロックと回避から始まります。回避には通常2つのチャージがあり、短いクールダウンで動作します。アクションバーの左側にはブロックゲージが表示されます。ブロックを開始するとこのゲージの一部を消費します。しかし、敵の攻撃に合わせて完璧なタイミングでブロックすると、**パリィ**が発生します。パリィに成功すると、多くの敵をよろめかせるだけでなく、ブロックゲージが大幅に速く回復します。

### リソースシステム：ディヴァイン・ファーバー（神聖なる熱意）
Fareverでは、マナ管理のストレスがありません。代わりに、クレリックは**ファーバー（Fervor）**と呼ばれるリソースを使用して行動します。武器スキルやコンボの最終段はファーバーを消費して、強力な追加効果を発動します。ファーバーは時間の経過とともにゆっくり回復しますが、激しいボス戦では特定の「ジェネレーター（生成）」スキルを使用してメーターを高く保つ必要があります。

| メカニクス | 機能 | クレリックへのメリット |
| :--- | :--- | :--- |
| **回避** | 無敵フレーム移動 | 回避不能な重攻撃を避けるために不可欠。 |
| **ブロックゲージ** | ダメージ軽減 | バフを付与するために近接範囲に留まることを可能にする。 |
| **パーフェクトパリィ** | よろめき＆回復 | ブロックゲージを補充し、回復のチャンスを作る。 |
| **ファーバーメーター** | スキルリソース | 「ベネディクション（祝福）」や上位の聖なる呪文の動力源。 |

## クレリックのパッシブ：ディヴァイン・フェイバー（神の加護）

Fareverの各クラスには独自のパッシブメカニクスがあります。クレリックの場合は**ディヴァイン・フェイバー**です。このシステムでは、ファーバーを一定のしきい値以上消費するたびに発動する「祝福（Blessing）」（メイジのコンジットに似たもの）をセットできます。

ファーバーバーの中央に切り欠き（ノッチ）があるのがわかります。メーターが50%以上の状態で武器スキルやコンボ最終段を使用すると、セットした祝福が発動します。最初は2つのスロットがあり、レベル6前後で3つ目が解放されます。

### おすすめの祝福セットアップ
*   **ラディアント・シャード（輝く破片）：** ターゲットに聖なるエネルギーの矢を放ちます。安定したDPSに最適です。
*   **ハロウド・グラウンド（聖域）：** 足元に小さな回復の光の領域を作り出します。
*   **ディヴァイン・マイト（神の力）：** 15秒間、物理および聖属性の熟練度を一定割合増加させます。最大20回までスタック可能です。

> **💡 ヒント：** ソロでのレベル上げには、ラディアント・シャード2つとディヴァイン・マイト1つの構成がおすすめです。これにより、ダメージバフを維持しながら、敵を倒すのに十分な火力を確保できます。

## クレリックの必須武器

レベル5になると、最初のレア（青）武器を選択できるミステリーキャッシュを受け取ります。ここでの選択が、今後数レベルのプレイスタイルを決定します。

### アービターズ・メイス（聖/物理）
これは標準的な「バトルクレリック」の選択肢です。敵に「サンクティファイド（神聖化）」デバフを付与する重い物理攻撃に焦点を当てています。サンクティファイド状態の敵にコンボ最終段を当てると、聖なる光の爆発が起こり、周囲の味方を回復します。

### レリクス・オブ・ザ・マーター（純サポート）
レリック（遺物）は、味方とリンク（テザー）することができる遠距離武器です。リンク中、自分が与えたダメージの一部がリンク先のターゲットの回復に変換されます。グループプレイやダンジョン攻略において最高の選択肢となります。

### レベル7のアーセナルシステム
レベル7に到達すると、**アーセナルスロット**が解放されます。これにより、サブ武器を装備し、ホットバーを切り替えることなくその武器固有のアビリティの1つを使用できるようになります。例えば、メイスを使用しているクレリックがアーセナルスロットにレリックを装備すれば、前線での存在感を維持しつつ、素早い緊急回復にアクセスできます。

| 武器タイプ | 主要ステータス | 最適な使用例 |
| :--- | :--- | :--- |
| **メイス** | 筋力 / 聖属性 | ソロのレベル上げと前線でのタンク役。 |
| **レリック** | 知力 / 聖属性 | 専任ヒーラーとグループのユーティリティ。 |
| **盾** | 生命力 | 最大の生存性とパリィウィンドウの確保。 |

## スキルとスキルルーン

レベル30に向かって進むにつれて、**スキルルーン**を入手することになります。これらはボスやワールドイベントからドロップするアイテムで、クラスのコアスキルをカスタマイズできます。スキルツリー内で手動でクリックして有効にする必要があります。

### 主要なクレリックスキル
1.  **ホーリービーム（レベル1）：** 主要なファーバー生成スキルです。ダメージを与え、ヒットごとにファーバーを回復するビームを放ちます。
2.  **イージス・オブ・ライト（レベル4）：** 最大体力の40%に相当するシールドを付与します。主要な防御ツールです。
3.  **ベネディクション（レベル10）：** 2分間のクールダウンを持ち、15秒間すべてのスキルのクールダウンを30%短縮します。
4.  **リザレクション（レベル20）：** 戦闘中に倒れた味方を蘇生させます。5分という長いクールダウンがあります。

> **⚠️ 警告：** ボスと戦う前には必ずイージス・オブ・ライトを発動させておきましょう。シールドは60秒間持続しますが、クールダウンは30秒なので、タイミングを合わせれば防御を「二重にスタック」させることができます。

## 進行とレベル上げのヒント

Fareverでクレリックを育てるには、攻撃的な戦闘と戦術的な撤退の組み合わせが必要です。伝統的なマナプールがないため、完全に「戦闘不能」になることはありませんが、呪文の合間に通常攻撃を挟まないと「ファーバー不足」に陥ることがあります。

1.  **最終コンボをマスターする：** 通常攻撃の4打目が「コンボ最終段」となります。これが「祝福」を発動させるトリガーです。可能な限り、通常攻撃のコンボを途中でキャンセルしないようにしましょう。
2.  **早めに武器をアップグレードする：** Fareverの武器には独自のレベルがあります。武器を使用するとランクが上がり、その武器固有のスキルを強化できます。ランク3のメイスは、より高いレア度のランク1のメイスよりも大幅に強力です。
3.  **環境を利用する：** すべてのものに物理的な当たり判定があるため、ホーリービームをチャージしている間、地形を利用して遠距離攻撃者の視線を遮ることができます。

## FAQ（よくある質問）

**Q: クレリックはFareverでソロクラスとして通用しますか？**

A: もちろんです。メイス武器とディヴァイン・マイトの祝福に焦点を当てることで、この**Farever クレリックガイド**のビルドは、ほぼすべてのワールドコンテンツをソロで攻略するのに十分な自己回復とダメージ軽減を提供します。

**Q: スキルルーンはどうすればもっと手に入りますか？**

A: スキルルーンは、プライムバレーのダンジョンボスやエリートモンスターから稀にドロップします。また、「アコライト・オブ・ライト」の陣営で「名誉（Honored）」ステータスに達すると、特定の評判ベンダーから購入することもできます。

**Q: クレリックはマナを使用しますか？**

A: いいえ。Fareverのすべてのクラスと同様に、クレリックはクラス固有のリソース（ファーバー）によって補完されたクールダウンベースのシステムを使用します。戦闘の合間に水やマナポーションを飲む心配はありません。

**Q: クレリックにとって最も重要なステータスは何ですか？**

A: バランスの取れたビルドにするには、回復量と呪文ダメージを増やすために**聖属性熟練度（Holy Mastery）**を、そしてイージス・オブ・ライトのシールド強度を高めるために**生命力（Vitality）**に注目してください。

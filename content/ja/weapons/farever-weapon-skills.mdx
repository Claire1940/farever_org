## Farever戦闘の核心的な仕組み

Fareverの戦闘は、伝統的なMMOのローテーションと「ソウルライク」なアクションのハイブリッドと表現するのが最適です。世界のすべての敵には物理的なヒットボックスがあります。アニメーションがキャラクターの占有する空間を通過すれば、ダメージを受けます。逆に、手動での移動やタイミングの良い回避が、主要な防御手段となります。

ジャンルの慣習からの最も大きな逸脱は、マナが完全に存在しないことです。代わりに、リソースは以下を通じて管理されます。
*   **クールダウン：** 回避やブロックを含むすべての有効なアビリティは、タイマーで作動します。
*   **スパークメーター：** メイジクラス固有のリソースで、追加のダメージやユーティリティを提供するパッシブ「コンジット（導管）」の動力源となります。
*   **ブロックゲージ：** ブロックを維持すると減少するスタミナのようなバーです。「パーフェクトブロック」を行うとパリィが発生し、このゲージがより速く回復します。

### 武器スキル vs. クラススキル

アクションバーは2つの明確な半分に分かれています。左側には**Fareverの武器スキル**があり、これは現在メインハンドに装備しているアイテムによって完全に変化します。これには、基本的な通常攻撃のコンボチェーンと、2つまたは3つのユニークなアクティブアビリティが含まれます。バーの右側にはクラススキルがあり、これらは装備に関係なく、キャラクターのレベルアップに合わせて一定のまま維持されます。

| スキルタイプ | 出所 | 目的 |
| :--- | :--- | :--- |
| **通常攻撃** | 装備中の武器 | 基本ダメージ。「ファイナルコンボアタック」で終了。 |
| **武器スキル** | 装備中の武器 | 影響力の高い能力（例：火炎放射器、トーテム）。 |
| **クラススキル** | キャラクターレベル | ユーティリティとコアローテーション（例：ブリンク、スパークシールド）。 |
| **アーセナルスキル** | サブ武器 | 別の武器タイプから引き出された単一のアビリティ。 |

## スパークマスター・パッシブの理解

メイジにとって、スパークマスターの仕組みは装備と生の魔力をつなぐ架け橋です。武器スキルを使用するか、「ファイナルコンボアタック」（通常攻撃チェーンの最後の一撃）を完了すると、スパークを消費します。

> **💡 エキスパートのヒント：** スパークメーターは常に半分以上のしきい値を維持するようにしてください。ゲージが半分以上溜まっている状態でスパークを消費すると、自動的に「コンジット」が発動します。これは魔法の破片やパワーバフなどの無料の追加呪文です。

スパークマスターのインターフェースに異なるコンジットをスロットすることで、パッシブ出力をカスタマイズできます。2026年の人気のセットアップでは、魔法ダメージを蓄積するための「パワー」コンジットを1つ、継続的な飛び道具の圧力を維持するための「シャード（破片）」コンジットを2つ運用します。

## アーセナルシステム：ロードアウトの構築

レベル7で解放されるアーセナルシステムは、おそらく**Fareverの武器スキル**カスタマイズにおいて最も柔軟な要素です。これにより、サブ武器を装備し、そのアビリティの1つを「借用」してメイン武器のキットと一緒に使用することができます。

例えば、ファイアスタッフの遠距離からの破壊力を楽しみつつも、敵が接近した際に脆弱だと感じる場合、アーセナルスロットに「拳（フィスト）」武器を装備できます。拳の「アッパーカット」スキルを選択することで、ファイアスタッフのパッシブボーナスを失うことなく、瞬時に近接攻撃に切り替えて敵を引き離すことができます。

### 試してみるべきアーセナルのシナジー
1.  **スタッフ + トーテム：** ダメージ用にファイアスタッフを使用しつつ、パッシブ回復用に「ゴーストクラム」のウォータートーテムを装備する。
2.  **魔導書 + ウェーブ：** 基本的なスペルブックのキットに、大規模な範囲（AoE）ノックバックを追加する。
3.  **拳 + ファイアブラスト：** 高機動の近接戦闘と即時発動の火炎バーストを組み合わせた「バトルメイジ」プレイスタイル。

## スターター武器の解説

レベル5に達すると、プレイヤーは通常、最初の主要な武器の選択肢が入ったブルーミステリーキャッシュを受け取ります。適切なものを選択するかどうかは、好みの交戦距離とユーティリティのニーズによって決まります。

### ラディアント・スタッフ（火）
ラディアント・スタッフは、大量の範囲ダメージを求めるプレイヤーにとって最高の選択肢です。その主要な仕組みは、**フレイムボム**パッシブを中心に展開します。

| スキル | 効果 | 主要なアップグレード |
| :--- | :--- | :--- |
| **ブレイジング・ビーム** | 継続的なダメージを与えるチャネル式の火炎放射。 | ランク3：すべてのティックが確定でクリティカルヒットになる。 |
| **パイロラズム** | 敵が受ける魔法ダメージを増加させる巨大な爆発。 | ランク3：クリティカル時に10%の確率でクールダウンがリセットされる。 |
| **フレイムボム** | パッシブ：敵に蓄積し、10スタックで爆発する。 | ランク1：必要なスタック数を15から10に減少させる。 |

ここでのシナジーは驚異的です。「ブレイジング・ビーム」は常にクリティカルになるようアップグレードでき、クリティカルは「フレイムボム」のスタックを適用するため、ビームを1回照射するだけで数秒の間に複数の爆発を引き起こすことができます。

### 低潮のゴーストクラム（水）
サポート役や高い生存性を好むなら、ゴーストクラムの右に出るものはありません。これらの「光輪（ハロ）」により、味方を回復し敵を減速させるトーテムを召喚できます。

| スキル | 効果 | 戦略的活用 |
| :--- | :--- | :--- |
| **タイダル・トーテム** | 2秒ごとに4-5 HPを回復するトーテムを設置。 | 範囲内に留まり、エリートモブに対して粘り強く戦う。 |
| **リップタイド** | 高い魔法ダメージを与える前方扇状の水バースト。 | トーテムの「スロウ」の後に使用し、25%のボーナスダメージを与える。 |
| **潮流の力** | パッシブ：スキルを使用するとトーテムが噴火し、範囲ダメージを与える。 | クラススキル（ブリンクなど）を織り交ぜてトーテムの噴火を誘発する。 |

## メイジのクラススキルと進行

**Fareverの武器スキル**が主要なダメージ源である一方、クラススキルはこのゲームの困難なダンジョンを生き抜くために必要なユーティリティを提供します。2026年現在、メイジのレベル進行により、大規模なバーストウィンドウを可能にするいくつかの「リセット」メカニクスが解放されます。

### 必須のメイジアビリティ
*   **スパーク・ビーム (Lv 1)：** 主要なスパーク生成手段。メーターが50%を下回ったら、あらゆるローテーションにこれを織り交ぜるべきです。
*   **スパーク・シールド (Lv 3)：** 最大体力の40%に相当するダメージを吸収します。クールダウンは30秒ですが持続時間は60秒であるため、ボス戦に入る前にシールドを「二重に重ねる」ことができます。
*   **ブリンク (Lv 5)：** 短距離テレポート。物理的なヒットボックスを回避し、敵の予備動作中に位置を調整するために使用します。
*   **クロノ・リセット (Lv 20)：** スパークを完全に回復し、すべての武器スキルのタイマーをリセットする強力な3分間のクールダウンアビリティ。

### ルーンのカスタマイズ
すべてのクラススキルは、ドロップアイテムとして見つかる「ルーン」を使用して変更できます。例えば、スパーク・シールド用の**リサージェンス・ルーン**は、シールドが破壊されたときに術者のスパークを回復します。ルーンは消費しても自動的には装備されないため、常にスキルメニュー（Kキー）を確認して、ルーンが有効に選択されているか確認してください。

> **⚠️ 警告：** ルーンはスペルブックへの恒久的な追加となりますが、1つのスキルにつき一度に1つしか有効にできません。現在の武器の弱点を補完するものを選んでください。

## 高度なローテーション戦略

Fareverのエンドゲームで秀でるためには、「ウィービング（織り交ぜ）」テクニックをマスターしなければなりません。これには、ファイナルコンボアタックまで通常攻撃チェーンを使用し、直後に武器スキルで硬直アニメーションをキャンセルし、続いてスパーク・ビームを使用して消費したリソースを補充するという一連の流れが含まれます。

ビルド固有の装備に関する詳細については、[Farever公式Steamコミュニティ](https://steampowered.com)でプレイヤーによるガイドやパッチノートをチェックしてください。

### ラディアント・スタッフのローテーション例：
1.  **パイロラズム：** ターゲットに8%の魔法ダメージデバフを付与。
2.  **ブレイジング・ビーム：** フルチャネルしてフレイムボムを蓄積し、ターゲットを減速させる。
3.  **ファイナルコンボ通常攻撃：** 敵が減速している間にコンジット（シャード/パワー）を発動させる。
4.  **スパーク・ビーム：** 前の手順で消費したスパークを回復。
5.  **即時パイロラズム：** ビームのクリティカルでクールダウンがリセットされた場合、すぐに再度発射する。

## FAQ

**Q: どうすればもっと多くのFarever武器スキルを解放できますか？**

A: 武器スキルは武器自体に紐付いています。より強力なバージョンを解放するには、戦闘でその武器を使用して武器レベルを上げる必要があります。ブルーティアの各武器には、武器のレベルが上がるにつれてアップグレードできる3つのランクが各スキルに用意されています。

**Q: メイジとして近接武器スキルを使用することはできますか？**

A: はい。レベル7で解放されるアーセナルシステムを通じて、サブスロットに拳や爪などの武器を装備し、その物理スキルを1つ使用することができます。これは、近接攻撃と魔法を混ぜ合わせたい「バトルメイジ」ビルドにとって非常に効果的です。

**Q: スパークを生成する最良の方法は何ですか？**

A: スパークを生成する最も信頼できる方法は、クラススキルの**スパーク・ビーム**を使用することです。ビームの各ティックで12のスパークが回復します。スパークは非戦闘時にパッシブで回復しますが、継続的なダメージを出すためには、アクティブなローテーションにスパーク・ビームを織り交ぜることが必須です。

**Q: 武器スキルにマナは必要ですか？**

A: いいえ、Fareverはマナシステムを使用しません。すべての**Fareverの武器スキル**とクラススキルは、特定のクールダウンタイマーと、メイジの場合はパッシブコンジットをトリガーするためのスパークの有無によってのみ制限されます。

<Section icon={Shield} title="プリースト：神聖なる汎用性">

プリーストは、おそらく **farever クラスティアリスト**の中で最も柔軟なクラスです。セプターから始まり、攻撃的なプレッシャーと命を救うサポートのバランスに優れています。独自の「ロザリオ」パッシブにより様々な祈りをサイクルさせ、それを特殊能力「審判」で解放することで爆発的な効果を発揮します。

| スキル名 | レベル | 効果 | 用途 |
| :--- | :--- | :--- | :--- |
| **信仰の風** | 初期 | 移動速度の大幅な範囲バフ | 配置/カイト |
| **輝く裁定** | 初期 | 長時間スタンを伴う範囲ダメージ | クラウドコントロール |
| **熱狂** | 10 | パーティ全体のダメージと回復量増加 | グループバフ |
| **クルセイダー** | 15 | 自身のダメージとCDRを強化するバフ | ソロDPS |
| **希望の灯火** | 20 | ダメージ/回復を蓄積し後に爆発させる | バースト回復/ダメ |
| **ミラクル** | 30 | 「審判」のCDR 50%と無料の祈り | エンドゲームの力 |

プリーストのタレントツリー（タンク、DPS、またはヒーラー）により、パーティ構成のあらゆる隙間を埋めることができます。レベル30の「ミラクル」状態では、15秒ごとに無料の祈りを与え、最も強力なアビリティのクールダウンを半分にする強力な存在となります。

<Callout type="success" title="なぜプリーストを使うのか？">
チームの屋台骨でありながら、困難なコンテンツをソロで攻略するツールも持ちたいのであれば、プリーストは最初のキャラクターとして最適な選択です。
</Callout>

</Section>

<Section icon={Swords} title="ウォーリアー：止められない力">

ウォーリアーは決定的な前線のスペシャリストです。剣と盾を装備し、攻撃を吸収し、ノックバックやスタンを通じて戦場をコントロールするように設計されています。「レイジ（怒り）」メカニクスは攻撃的なプレイに報酬を与え、攻撃するたびに壊滅的な「レイジングスマッシュ」に必要なリソースが蓄積されます。

| アビリティ | タイプ | 主な利点 |
| :--- | :--- | :--- |
| **なぎ払い** | 範囲 | ダメージを与え、ノックバックさせる |
| **シールドバッシュ** | CC | 確実な単体スタン |
| **苦痛耐性** | 防御 | **40%のダメージ軽減**（ゲーム内最強） |
| **バトルシャウト** | バフ | パーティのクリティカル率を20%増加 |
| **バーサーク** | 強化 | レイジ生成の加速とダメージ上昇 |
| **怒りの爆発** | ユーティリティ | 次の3つの武器スキルのクールダウンを削除 |

ウォーリアーのタレントツリーは、クリティカルフォーカス、ヘビーブリード、または純粋なタンクに焦点を当てています。「苦痛耐性」を使用できるため、高圧的な状況でもほぼ不死身であり、2026年の **farever クラスティアリスト**のトップの座を確保しています。

<Callout type="warning" title="レイジ管理">
初心者のウォーリアーはレイジを溢れさせてしまいがちです。「レイジングスマッシュ」を頻繁に使用して範囲ダメージを最大化し、敵を怯ませ続けましょう。
</Callout>

</Section>

<Section icon={Sparkles} title="メイジ：スパークの達人">

メイジは複雑ながらもやりがいのある遠距離プレイスタイルを提供します。「スパークマスター」パッシブを利用して、メイジはスパークスロットを入れ替えることで呪文の効果をカスタマイズできます。これにより、クラウドコントロールが必要な場合でも、純粋な属性ダメージが必要な場合でも、高い適応性を発揮します。

| 主要スキル | 説明 | 戦略的使用 |
| :--- | :--- | :--- |
| **パイロボール** | 大ダメージと燃焼デバフ | 初撃のバースト |
| **ファイアブラスト** | 燃焼中の対象に即時ダメージとスタン | 呪文の中断 |
| **ブリンク** | 方向指定テレポート | ボスのギミック回避 |
| **神秘のエンパワーメント** | パーティのCDRバフ | チームDPSの最大化 |
| **スタティックノヴァ** | 範囲ダメージと移動不能 | 雑魚敵の制御 |
| **クロノリセット** | スパークとアビリティを再生 | 継続戦闘 |

レベル30で、メイジは「オーバーロード」をアンロックし、1秒ごとにランダムなスパークを生成します。これにより、混沌としていながらも極めて高いダメージウィンドウが生まれ、戦場を支配できます。タレントツリーにはスパークフォーカス、マジックブラスター、そして驚くべきことにヒーリングツリーも含まれており、単なるガラスの大砲以上の存在となります。

<Callout type="info" title="スパークのカスタマイズ">
ダンジョンに入る前に必ずスパークマスターの設定を確認してください。敵の弱点に合わせてスパークを入れ替えることが、プロのメイジプレイヤーの証です。
</Callout>

</Section>

<Section icon={Target} title="ローグ：影と鋼">

ローグは、機動力と精密さを重視するプレイヤーにとって最高の選択肢です。短剣から始まり、コンボポイントシステムを利用して「フィニッシャー」アビリティを強化します。そのキットにはテレポートやユーティリティスキルが満載されており、敵の防御を回避して脆弱なターゲットを攻撃することができます。

<Steps>
  <Step num="1" title="コンボポイントの生成">
    基本攻撃のフィニッシャーや「テンペストナイフ」などのアビリティを使用してリソースバーを溜めます。
  </Step>
  <Step num="2" title="「死の刻印」の付与">
    レベル10で、このクールダウンを使用してターゲットへの全ダメージを30%増加させます。
  </Step>
  <Step num="3" title="フィニッシャーの解放">
    全ポイントを消費して大ダメージを与えます。レベル20のパワースパイク中、フィニッシャーは10秒間クールダウンがなくなります。
  </Step>
  <Step num="4" title="ユーティリティの活用">
    「スモークボム」を展開してパーティの回避を助け、「シャドウステップ」で位置を調整します。
  </Step>
</Steps>

ローグのタレントツリーは、毒、クールダウン短縮、またはユーティリティの道を提供します。ウォーリアーよりも繊細な操作が必要ですが、レベル30で影を召喚する能力（ダークネスシャドウズ）により、継続ダメージ出力が大幅に向上します。

<Callout type="tip" title="バックスタブボーナス">
「シャドウジャンプ」のようなスキルは自動的に敵の背後に回り込みます。ローグ本来の利点を最大限に活かすため、常に背後からの攻撃を優先しましょう。
</Callout>

</Section>
